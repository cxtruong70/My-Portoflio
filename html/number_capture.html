Number Capture

Phone App Feature

Introduction

This was a design challenge for my UI design class. We were tasked with individually designing a new feature for an existing application. Using Adobe XD and my sketchbook I designed this feature, created a mock-up and an interactive prototype.

Feature Description

This design feature allows users to input phone numbers directly from the physical world into the iOS phone application: whether it be from billboards, credit cards, or a new restaurant you are passing by.
